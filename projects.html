Unix Utilities

This series of projects was completed in spring 2019 as part of the COMP590 Systems Programming course, though I added additional functionality and extensions along the way that were not required for coursework. They are all command-line applications written in Rust, and details about each are given below:

egrep

On this project, I worked with a peer, breaking the program into stages of tokenization, parsing, construction of an NFA, and use of that NFA to test input strings against a regular expression. We worked in a ping-pong pair-programming approach with test-driven development. There is support for concatenating NFAs, and it is also possible to view DOT visualizations of the created NFAs. Supported operations include concatenation, alternation (|), kleene star (*), any character (.), and kleene plus (+).

bc

I built up the Basic Calculator as a recursive descent parser. It takes in mathematical expressions in infix notation and emits the equivalent postfix notation expression to be used with the dc program in order to then perform the calculation. Additionally, I added support for exponentiation and register assignment.

dc

For the Desk Calculator, I tokenized input that came in the form of reverse-polish notation mathematical expressions then implemented a stack machine to process the tokens and carry out their instructions. Support for exponentiation and registers was also included.

echo

This implementation includes support for select escape sequences and other options based on flags passed in by the user.
